[메인]

우측에는 인사 멘트와 연결되는 말풍선을 배치하여 챗봇 상담으로 연계될 수 있도록 하고, 중앙에 메인 기능인 테라피 메뉴로 바로 연결될 수 있는 박스형 버튼을 배치하여 쉽고 빠르게 접근할 수 있도록 하였습니다. 또한, 중앙 상단 버튼 아래로 부가서비스를 슬라이드 형태로 배치하여서 한 화면에서 APP의 모든 주요 메뉴에 접근할 수 있도록 하였습니다.

[상담 챗봇, 마음이]

채팅 내에서 부정 감정과 긍정 감정을 컬러로 구분하여 더욱 쉽게 선택할 수 있도록 하였습니다.
메인과 마찬가지로 페이지의 이동 없이 한 페이지 안에서 APP의 주요한 기능을 최대한 사용할 수 있도록 챗봇과의 대화 결과에 따라 추천된 음악을 채팅창 내에서 바로 재생할 수 있도록 구성하였습니다.
